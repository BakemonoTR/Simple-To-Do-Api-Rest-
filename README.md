# Simple-To-Do-REST-API
Simple To-Do Task Manager



FastAPI, SQLModel ve SQLite kullanılarak oluşturulmuş basit bir Yapılacaklar Listesi API'si.

Bu proje, CV portfolyosu için 1 günlük bir sprint kapsamında geliştirilmiştir.

## Kullanılan Teknolojiler
- **Python 3.10+**
- **FastAPI:** Modern ve yüksek performanslı web framework'ü.
- **SQLModel:** Veritabanı modelleri ve API doğrulaması için.
- **SQLite:** Sunucu gerektirmeyen SQL veritabanı.
- **Uvicorn:** ASGI sunucusu.

## Özellikler
- Tam CRUD (Create, Read, Update, Delete) işlemleri.
- Otomatik interaktif API dokümantasyonu (Swagger UI & ReDoc).
- Modern `lifespan` eventleri ile veritabanı yönetimi.
- Kalıcı veri depolama.
